HLS 家庭内LANでブラウザで視聴できます。
対応ブラウザ Microsoft Edge, MAC Safari
URL rtmp Youtube や AWS MediaLive へ配信ができます。
e.g. rtmp://192.168.1.1/aaa/bbb -> url=rtmp://192.168.1.1/aaa, key=bbb
URL srt Listener srt://:3000
Caller srt://(YourPC):3000
カメラ カメラサイズ。(960x540, 1280x720)
自動低画質 RTMP配信が間に合わないときビットレートを自動的に下げます。
自動停止 配信を自動的に停止します。
ビットレート １秒間のデータ量。(Kbps)
フレームレート １秒間に描画する枚数。(fps)
顔ボタン 顔を認識して目線を隠します。
